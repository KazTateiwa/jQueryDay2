Username and Password Checker

Username and Password Requirements:

  1. User ID and password cannot be the same
  2. User ID and password have to be at least six characters long
  3. User ID cannot contain the following characters: !#$
  4. Password has to contain at least one of: !#$
  5. Password cannot be "password"
  6. Password has to contain at least one digit
  7. Password has to contain a lower and an uppercase letter (hint: use toUpperCase or toLowerCase)
